# About

Hi — I’m Dusty.

I work in analytics and data modeling, with a background in actuarial science and a growing focus on data engineering. I enjoy building structured datasets from messy real-world sources and turning them into something measurable and useful.

As a competitor and coach, I started this project with a simple question: what does the competitive landscape of modern jiu-jitsu actually look like when you zoom out? I wanted to 
use my technical background to make that landscape more transparent — to better understand how the sport is structured, how athletes progress, and hopefully make competing feel 
a little less intimidating for newer competitors.

On the technical side, this project includes:

- Python-based data collection
- Cloud infrastructure for long-running scraping and Automation
- SQL modeling with dbt
- Structured data warehousing
- Analytical modeling and visualization

I’m particularly interested in competitive modeling — rating systems like ELO, predictive performance modeling, and how we can better measure skill progression over time.

If you’re interested in the technical side of the project or want to connect, you can find me on GitHub and LinkedIn.
